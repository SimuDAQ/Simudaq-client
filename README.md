# Simudaq

## 소개

Simudaq은 실제 한국 주식 시장 데이터를 활용한 모의 투자 서비스입니다. 초보 투자자부터 숙련된 트레이더까지, 위험 부담 없이 투자 전략을 연습하고 검증할 수 있는 환경을 제공합니다.

## 기술 스택

- **Vite** 
- **TypeScript** 
- **React** 
- **shadcn/ui** 
- **Tailwind CSS** 
- **React Query**
- **React Router** 
- **Recharts** 


### 사용 가능한 명령어

```sh
npm run dev          # 개발 서버 시작
npm run build        # 프로덕션 빌드
npm run build:dev    # 개발 모드 빌드
npm run lint         # 코드 린팅
npm run preview      # 빌드 결과 미리보기
```
